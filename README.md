# 股票交易游戏 (Stock Trading Game)

一个两人对战的股票交易模拟小游戏。

## 游戏特性

- **游戏设置**: 可配置初始股价、游戏天数、玩家初始资金、股价涨跌概率
- **股票模拟**: 每天股价按设定概率进行10%的涨跌
- **双人对战**: 两位玩家使用相同的初始资金进行竞争
- **交易系统**: 支持按股数或金额买入/卖出股票
- **可视化**: 股票走势图和玩家资产变化图

## 如何运行

1. 直接在浏览器中打开 `index.html` 文件
2. 或者使用本地服务器（推荐）：

```bash
# 使用 Python
python -m http.server 8080

# 使用 Node.js
npx serve .
```

然后访问 `http://localhost:8080`

## 游戏规则

1. 在设置页面配置游戏参数：
   - 初始股价（默认：100）
   - 游戏天数（默认：30）
   - 玩家初始资金（默认：10000）
   - 上涨概率（默认：50%）

2. 点击"开始游戏"进入游戏

3. 游戏进行中：
   - 每天玩家可以买入或卖出股票
   - 买入卖出可以按股数或按金额计算
   - 点击"进入下一天"推进游戏
   - 股价每天会按概率进行10%的涨跌

4. 游戏结束时，总资产更高的玩家获胜

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- SimpleChart - 自定义折线图库（无外部依赖）
